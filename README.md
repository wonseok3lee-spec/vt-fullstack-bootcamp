# Virginia Tech × Fullstack Academy — Data Analytics Bootcamp

This repository contains my practice projects and coursework from the  
[Virginia Tech Data Analytics Bootcamp](https://bootcamp.cpe.vt.edu/programs/data-analytics), delivered in partnership with Fullstack Academy.

Each unit includes hands-on assignments focused on applying data analytics concepts using real datasets.

For more detailed write-ups and reflections, see my blog:  
[Pecunia Tech](https://wonseok3lee-spec.github.io/)

---

## About the Program

A structured program covering the core data analytics stack — including Excel, SQL, Python, data visualization, and cloud fundamentals — with project-based assignments and career simulations.

---

## Curriculum & Tools

| Unit | Focus | Tools |
|------|-------|-------|
| Unit 1 | Business Analytics | Excel (PivotTables, Power Query, What-If Analysis) |
| Unit 2 | Data Acquisition & Manipulation | SQL (Joins, CTEs, Views, Window Functions) |
| Unit 3A | Cloud & ETL | AWS (Cloud Foundations, Glue) |
| Unit 4 | Data Visualization | Tableau (Calculated Fields, Dashboards) |
| Unit 5 | Programming Foundations | Python (Data Structures, Functions, Loops) |
| Unit 6 | Data Analytics with Python | Pandas, NumPy, Statistics |
| Elective | Statistical Analysis & ML | R (tidyverse, regression, clustering) |
| Elective | Business Intelligence | Power BI (DAX, dashboards) |
